# <LocationForm display toggleDisplay hideDisplay obtainData featureProperties collectionModel />

A component for displaying a form to add new locations to be tracked.

PropTypes:
Name Type Required Description Default
display bool Yes Determines whether the form for entering location name, longitude, and latitude values is displayed.
toggleDisplay func Yes Toggles whether the form is displayed. If the remove form is displayed, it will be hidden.
hideDisplay func Yes Called when a location has been successfully added to hide the form.
obtainData func Yes Called when a location has been successfully added to obtain the updated location list from the server and add the location to the map.
featureProperties object No
collectionModel object No

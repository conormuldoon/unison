Constructor

<ChartPopup varCur location fromDate toDate closePopup />

A popup that displays a TabsComponent. Once mounted, it connects to the back-end to obtain data for the from date, to date, location, and weather variable selected.

PropTypes:
Name Type Required Description Default
varCur string Yes Specifies the weather variable currently selected.
location string Yes Sepecifies the location selected.
fromDate string Yes Specifies the start date for the data that is to be displayed.
toDate string Yes Specifies the end date for the data that is to be displayed.
closePopup func Yes Called when the close icon is clicked.

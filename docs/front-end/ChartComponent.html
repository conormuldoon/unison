# <ChartComponent zoomDomain handleZoom data curVar minMax index />

A component for displaying line graphs.

PropTypes:
Name Type Required Description Default
zoomDomain object No Specifies the beginning and end times of the time domain of the x-axis.
handleZoom func Yes Called when there is a zoom domain change.
data array Yes Data to be displayed on the graph.
curVar string Yes The weather variable currently selected.
minMax bool No True if the if displaying tabs for the precipitation variable.
index number No The tab index - 0 for the left tab, 1 for the middle tab, and 2 for the right tab.

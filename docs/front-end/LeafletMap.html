LeafletMap

<LeafletMap curVar fromDate toDate marker markerCallback mapCentre featureProperties linksProperty />

A component for displaying a Leaflet map and markers for popups for locations where weather data is being tracked.

Constructor

# <LeafletMap curVar fromDate toDate marker markerCallback mapCentre featureProperties linksProperty />

PropTypes:
Name Type Required Description Default
curVar string No Specifies the weather variable currently selected.
fromDate string No Specifies the start date for the data that is to be displayed.
toDate string No Specifies the end date for the data that is to be displayed.
marker array No An array of markers to be displayed on the map. Each element of the array specifies the coordinates and location name associated with a marker.
markerCallback func No A function called when a marker is clicked.
